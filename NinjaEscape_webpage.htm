About

Ninjia Escape game is inspire by PacMan. There are two characters are created for this game - Ninja and Enemy. This game is constructed by using object-oriented programming.

Program List:

  • C#
  • CSS
  • HTML
  • Python
The game graphics is build by using an unity 2D Software. Another, we created the scores like counting the coins, hearts also. Ninja needs to run away from the Enemy. Both the Ninja and Enemy needs an animation and movement. We can use the keyboard/ joystick conreoller to control direction of the player - left, right, up, down. we can say that this game is a typical classical video game which is focused on a game-play rather than a a content or story games.

How It Works

Game rules to Play:

1. Ninja

    Ninja run away from Enemy.
    Player can use keyboard or joystick to control the direction of Ninja â€“ left, right, up and down.
    If Ninja moves left or right, its face will be flipped.


2. Enemy
    Enemy follow the way point we set.
    We create the empty Game Object and named Waypoint. And then, put it in where you want the enemy move to on the maze.


3. Grading
    When Enemy reach the coin or hearts, they will disappear, and the score will increase.
    1 coin is counted 10 points.
    1 heart is counted 1 life point.


4. Death
    When enemy catch the Ninja, Ninja will be killed and the game start over again.
